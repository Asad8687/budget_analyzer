Programmet är ett enkelt CLI-baserat Pythonprojekt för att registrera och analysera utgifter. Den låter användaren att lägga till utgifter, visa alla utgifter och beräkna total kostnaden. Denna projektet är byggt med fokus på Git och venv.

Starta programmet

Klona repot:https://github.com/Asad8687/budget_analyzer.git
cd budget_analyzer
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt

När den virtuella miljön är aktiv, kör programmet med:py -m app.main


Programmet startar i terminalen och visar en meny där användaren kan:
Lägga till utgifter
Visa alla utgifter
Visa total kostnad
Avsluta programmet
